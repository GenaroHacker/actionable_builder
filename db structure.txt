ID: Unique identifier for the task.
Task Name: A descriptive name for the task.
Tags: Keyword for easy categorization and retrieval.
Priority: A ranking to indicate the importance of the task within its tag.
Duration: The estimated time needed to complete the task in minutes.
Dependency: A task ID that needs to be completed before the task can be started.
Start Time: The earliest time the task can be started.
End Time: The latest time the task can be finished.
Category: Type of the task (create, excersise, relax, sleep).
Frequency: How often the task needs to be performed in days.
