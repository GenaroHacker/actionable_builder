--CONTEXTS--
ID (int): Context's unique key.
NAME (string): Context name, e.g., 'Work', 'Exercise'.
EXPECTED_PERCENTAGE (int): Current week's expected time %.
FUTURE_EXPECTED_PERCENTAGE (int): Next week's expected time %.
R (float): Red color component, 0-1.
G (float): Green color component, 0-1.
B (float): Blue color component, 0-1.

--TAGS--
ID (int): Tag's unique key.
NAME (string): Tag name.
CONTEXT_ID (int): Foreign key to contexts table.

--TASKS--
ID (int): Task's unique key.
NAME (string): Task name.
CONTEXT (string): Task type, e.g., 'create'.
TAG_ID (int): Tag's ID.
RANK (int): Task rank within tag.
DURATION (int): Estimated minutes to complete.
DEPENDENCY (int): Prerequisite task ID.
START (datetime): Earliest start time.
END (datetime): Latest end time.
FREQUENCY (int): Task frequency in days.
WEEKDAYS (string): Specific weekdays, e.g., "yyyyynn".

--SUBTASKS--
ID (int): Subtask's unique key.
TASK_ID (int): Foreign key to tasks table.
ORDER (int): Subtask sequence.
NAME (string): Subtask name.
DETAILS (string): Subtask notes.
DURATION (int): Minutes to complete.

--PLACEHOLDERS--
ID (int): Placeholder's unique key.
TYPE (string): Category, e.g., "song".
VALUE (string): Placeholder content.

--HISTORY--
ID (int): History entry's unique key.
SUBTASK_ID (int): Foreign key to subtasks.
ACTION (string): Action type (Started, Paused, Resumed, Canceled, Completed.).
TIME (datetime): Action time.
